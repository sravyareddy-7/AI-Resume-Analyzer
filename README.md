# AI Resume Analyzer

An AI-powered Resume Analyzer built using Python and Streamlit.

## Features
- Upload PDF Resume
- Extract Resume Text
- Detect Technical Skills
- Calculate ATS Score
- Identify Missing Skills
- Provide Resume Improvement Suggestions

## Tech Stack
- Python
- Streamlit
- PyPDF2

## How to Run

pip install -r requirements.txt

streamlit run app.py
